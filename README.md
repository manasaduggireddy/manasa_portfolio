# 🌐 Manasa Duggireddy - Personal Portfolio

Welcome to my personal portfolio website! This portfolio showcases my skills, education, projects, and career aspirations as a Computer Science and Engineering student. It is designed with a modern, responsive, and interactive user interface to provide visitors with a seamless browsing experience.

## ✨ Features

* Modern and responsive design
* Interactive hero section with typing animation
* About Me section
* Education timeline
* Technical skills with animated progress bars
* Projects showcase
* Contact form
* Dark/Light mode toggle
* Smooth scrolling and animations
* Mobile-friendly layout

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Font Awesome
* Google Fonts
